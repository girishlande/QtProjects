This project shows how to create new process from your application. 
This will bind some function calls on process start and stop operation.
When process starts it will show messagebox , everything will get disabled.
when process ends it will destroy messagebox and everything is enabled again. 

For testing purpose "use subfolder timer project"
Upon building this subproject, it will create executable which take 5 seconds to run. 
You can call this process by passing appropriate path of the application
